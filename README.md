# Backend API

This project is a simple backend REST API built using Python and Flask.

## Features
- Create a room
- Get all rooms
- Get room by ID

## Tech Stack
- Python
- Flask
- Postman

## How to Run
1. Install Flask
2. Run `python main.py`
3. Open `http://127.0.0.1:5000`

## API Endpoints
- POST /rooms
- GET /rooms
- GET /rooms/<id>

## Testing
All APIs were tested using Postman.
